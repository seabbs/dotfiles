  Workflow Cheatsheet            /search  n/N next/prev  q quit
  ================================================================

  TMUX POPUPS (prefix = Ctrl-Space)
  ----------------------------------------------------------------
  Ctrl-Space G     GitHub dashboard (global)
  Ctrl-Space g     GitHub dashboard (current repo)
  Ctrl-Space A     AI agent popup (ephemeral, current dir)
  Ctrl-Space T     btop system monitor
  Ctrl-Space t     scratch terminal
  Ctrl-Space C-g   lazygit popup (current dir)
  Ctrl-Space C-h   this cheatsheet

  TMUX PANES
  ----------------------------------------------------------------
  Ctrl-Space |     split vertical
  Ctrl-Space -     split horizontal
  Ctrl-Space h/j/k/l    navigate panes
  Ctrl-Space H/J/K/L    resize panes
  Ctrl-Space z     zoom/unzoom pane
  Ctrl-Space b     break pane to new window
  Ctrl-Space J     join pane from another window

  TMUX WINDOWS
  ----------------------------------------------------------------
  Ctrl-Space c     new window
  Ctrl-Space D     dev layout (nvim | happy / repl)
  Ctrl-Space 1..9  switch to window N

  TMUX SESSIONS
  ----------------------------------------------------------------
  Ctrl-Space f     session switcher (fzf popup)
                 C-a all  C-s sessions  C-p projects  C-d kill
  Ctrl-Space \     toggle last session
  Ctrl-Space /     toggle last window
  Ctrl-Space s     session picker (built-in)
  Ctrl-Space w     window/session tree
  Ctrl-Space (/)   prev/next session
  Ctrl-Space r     reload tmux.conf

  TMUX URLS & TEXT GRABBING
  ----------------------------------------------------------------
  Ctrl-Space u     pick URL from pane (fzf, opens in browser)
  Ctrl-Space Tab   pick any text from pane (extrakto, inserts)

  TMUX SESSION SAVE/RESTORE
  ----------------------------------------------------------------
  Ctrl-Space C-s   save session layout (resurrect)
  Ctrl-Space C-r   restore session layout (resurrect)
                 continuum auto-saves every 15 min

  TMUX COPY MODE
  ----------------------------------------------------------------
  Ctrl-Space [     enter copy mode (scroll with j/k, C-u/C-d)
  v            begin selection
  y / Enter    yank to clipboard

  SHELL: PROJECT MANAGEMENT
  ----------------------------------------------------------------
  proj <name> [branch]     open full project (nvim + happy + repl)
  agent [name] [worktree]  open agent session (happy only)
  agent-to-proj            upgrade agent to full project layout
  gproj / gagent           gemini variants of proj / agent
  lsproj [filter]          list available projects
  gm                       checkout main and pull latest

  SHELL: WORKTREES
  ----------------------------------------------------------------
  feat <branch> [base]     new feature worktree window
  feat-done <branch>       cleanup worktree and window
  feat-list                list all worktrees
  agent-feat <branch>      agent worktree window
  agent-feat-done <branch> cleanup agent worktree

  SHELL: SESSION MONITORING
  ----------------------------------------------------------------
  projects     list active project sessions
  agents       list active agent sessions
  mtmux        list all sessions

  SHELL: TOOLS
  ----------------------------------------------------------------
  cat <file>       syntax-highlighted cat (bat)
  catp <file>      bat with pager
  bench a.R        benchmark a script (auto-detects R/jl/py)
  bench a.jl b.jl  compare two scripts
  job-status       show scheduled job status and logs
  sync-repos       fetch and fast-forward all repos

  SHELL: AI COMMANDS
  ----------------------------------------------------------------
  ai             direct CLI agent (no tmux)
  tc <cmd>       tmux + claude session
  commit         commit (haiku)
  review         code review
  pr             create PR (sonnet)
  lint           lint changed files (sonnet)
  test           run tests (sonnet)
  scan-issues    find fixable issues
  issue-summary  summarise an issue
  uk-news        BBC/Guardian summary

  GH DASH (inside Ctrl-Space G popup)
  ----------------------------------------------------------------
  s          cycle views (PRs / Issues / Notifications)
  h / l      switch section
  j / k      navigate items
  /          edit section filter
  p          open project for selected repo
  a          open agent for selected repo
  C          code review PR in Neovim (Octo)
  R          request reviewer on PR
  O          checkout PR branch
  o          open in browser
  d          mark notification done
  ?          full gh-dash help

  GIT DIFF (delta side-by-side pager)
  ----------------------------------------------------------------
  n            next file/hunk
  N            previous file/hunk
  q            quit pager
  /            search
  g            go to top
  G            go to bottom

  NEOVIM: SEND TO REPL (vim-slime → tmux bottom-right pane)
  ----------------------------------------------------------------
  <leader>rc     send line (normal) / selection (visual)
  <leader>rp     send paragraph
  <leader>rf     send whole file
  <leader>ru     send up to cursor line
  <leader>rF     send function (treesitter)
  <leader>rb     send code block (quarto/rmd)
  <leader>ra     send all code blocks (quarto/rmd)
  <leader>rU     send code blocks up to cursor (quarto/rmd)
  <leader>ro     start httpgd plot server (R)
  <leader>r=     reconfigure target pane

  NEOVIM: OTHER
  ----------------------------------------------------------------
  <leader>?      searchable keybinding cheatsheet (legendary.nvim)
  <leader>ci     GitHub Actions pipeline status (pipeline.nvim)
  \ll            start/stop LaTeX compilation (VimTeX)
  \lv            forward search to Skim PDF

  AEROSPACE (Alt = modifier, manages outer windows)
  ================================================================

  FOCUS / MOVE
  ----------------------------------------------------------------
  Alt-h/j/k/l          focus window
  Alt-Shift-h/j/k/l    move window
  Alt-Ctrl-h/j/k/l     join with adjacent container

  WORKSPACES
  ----------------------------------------------------------------
  Alt-1..6              switch workspace
  Alt-Shift-1..6        move window to workspace
  Alt-\                 toggle last workspace
  Alt-/                 toggle last focused window
  Alt-Shift-=           balance window sizes

  LAYOUT
  ----------------------------------------------------------------
  Alt-z                 fullscreen toggle
  Alt-Shift-/           cycle tile layouts (h/v)
  Alt-Ctrl-/            cycle accordion layouts (h/v)

  APP LAUNCHERS / FINDERS
  ----------------------------------------------------------------
  Alt-t                 focus/open Ghostty
  Alt-f                 fzf window finder
  Alt-w                 workspace tree view
  Alt-o                 open clipboard URL in browser

  SPLIT MODE (Alt-s then key)
  ----------------------------------------------------------------
  b                     split + browser
  g                     split + Ghostty
  o                     split + Obsidian
  Escape                cancel

  MONITORS
  ----------------------------------------------------------------
  Alt-.                 focus next monitor
  Alt-Shift-.           focus prev monitor
  Alt-Shift-Tab         move workspace to next monitor

  ACTIONS
  ----------------------------------------------------------------
  Alt-x                 close window
  Alt-Shift-c           close all but current window
  Alt-b                 break window to new workspace
  Alt-r                 resize mode (h/j/k/l, Esc to exit)
  Alt-Shift-;           service mode (r=reload, f=flatten)

  SCRATCHPAD (comms apps)
  ----------------------------------------------------------------
  Alt-Shift-w           toggle WhatsApp
  Alt-Shift-s           toggle Signal
  Alt-Shift-m           toggle Messages
  Alt-Shift-a           toggle Slack
  Alt-Shift-e           toggle Teams
  Alt-Shift-x           hide all scratchpad windows
  Alt-Shift-n           cycle next scratchpad window
