  Workflow Cheatsheet            /search  n/N next/prev  q quit
  ================================================================

  TMUX POPUPS (prefix = Ctrl-b)
  ----------------------------------------------------------------
  Ctrl-b G     GitHub dashboard (global)
  Ctrl-b g     GitHub dashboard (current repo)
  Ctrl-b A     AI agent popup (ephemeral, current dir)
  Ctrl-b T     btop system monitor
  Ctrl-b t     scratch terminal
  Ctrl-b C-h   this cheatsheet

  TMUX PANES
  ----------------------------------------------------------------
  Ctrl-b |     split vertical
  Ctrl-b -     split horizontal
  Ctrl-b h/j/k/l    navigate panes
  Alt-h/j/k/l        navigate panes (no prefix)
  Ctrl-b H/J/K/L    resize panes
  Ctrl-b z     zoom/unzoom pane
  Ctrl-b b     break pane to new window
  Ctrl-b J     join pane from another window

  TMUX WINDOWS
  ----------------------------------------------------------------
  Ctrl-b c     new window
  Ctrl-b D     dev layout (nvim | happy / repl)
  Alt-1..9     switch to window N

  TMUX SESSIONS
  ----------------------------------------------------------------
  Ctrl-b f     session switcher (fzf popup)
                 C-a all  C-s sessions  C-p projects  C-d kill
  Ctrl-b L     toggle last session
  Ctrl-b s     session picker (built-in)
  Ctrl-b w     window/session tree
  Alt-(/)      prev/next session
  Ctrl-b r     reload tmux.conf

  TMUX COPY MODE
  ----------------------------------------------------------------
  Ctrl-b [     enter copy mode
  v            begin selection
  y / Enter    yank to clipboard

  SHELL: PROJECT MANAGEMENT
  ----------------------------------------------------------------
  proj <name> [branch]     open full project (nvim + happy + repl)
  agent [name] [worktree]  open agent session (happy only)
  agent-to-proj            upgrade agent to full project layout
  gproj / gagent           gemini variants of proj / agent
  lsproj [filter]          list available projects
  gm                       checkout main and pull latest

  SHELL: WORKTREES
  ----------------------------------------------------------------
  feat <branch> [base]     new feature worktree window
  feat-done <branch>       cleanup worktree and window
  feat-list                list all worktrees
  agent-feat <branch>      agent worktree window
  agent-feat-done <branch> cleanup agent worktree

  SHELL: SESSION MONITORING
  ----------------------------------------------------------------
  projects     list active project sessions
  agents       list active agent sessions
  mtmux        list all sessions

  SHELL: TOOLS
  ----------------------------------------------------------------
  cat <file>       syntax-highlighted cat (bat)
  catp <file>      bat with pager
  bench a.R        benchmark a script (auto-detects R/jl/py)
  bench a.jl b.jl  compare two scripts
  job-status       show scheduled job status and logs
  sync-repos       fetch and fast-forward all repos

  SHELL: AI COMMANDS
  ----------------------------------------------------------------
  tc <cmd>       tmux + claude session
  commit         commit (haiku)
  review         code review
  pr             create PR (sonnet)
  lint           lint changed files (sonnet)
  test           run tests (sonnet)
  scan-issues    find fixable issues
  issue-summary  summarise an issue
  uk-news        BBC/Guardian summary

  GH DASH (inside Ctrl-b G popup)
  ----------------------------------------------------------------
  s          cycle views (PRs / Issues / Notifications)
  h / l      switch section
  j / k      navigate items
  /          edit section filter
  p          open project for selected repo
  a          open agent for selected repo
  C          code review PR in Neovim (Octo)
  R          request reviewer on PR
  O          checkout PR branch
  o          open in browser
  d          mark notification done
  ?          full gh-dash help

  GIT DIFF (delta side-by-side pager)
  ----------------------------------------------------------------
  n            next file/hunk
  N            previous file/hunk
  q            quit pager
  /            search
  g            go to top
  G            go to bottom

  NEOVIM: SEND TO REPL (vim-slime â†’ tmux bottom-right pane)
  ----------------------------------------------------------------
  <leader>sc     send line (normal) / selection (visual)
  <leader>sp     send paragraph
  <leader>sf     send whole file
  <leader>su     send up to cursor line
  <leader>sF     send function (treesitter)
  <leader>sb     send code block (quarto/rmd)
  <leader>sa     send all code blocks (quarto/rmd)
  <leader>sU     send code blocks up to cursor (quarto/rmd)
  <leader>so     start httpgd plot server (R)
  <leader>s=     reconfigure target pane

  NEOVIM: OTHER
  ----------------------------------------------------------------
  <leader>?      searchable keybinding cheatsheet (legendary.nvim)
  <leader>ci     GitHub Actions pipeline status (pipeline.nvim)
  \ll            start/stop LaTeX compilation (VimTeX)
  \lv            forward search to Skim PDF
